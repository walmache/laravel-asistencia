# Instrucciones para Agregar el Favicon

Para agregar el logo como favicon:

1. Coloca el archivo de imagen del logo en la carpeta `public/` con el nombre `favicon.png`
2. Formatos soportados: PNG, ICO, SVG
3. Tamaño recomendado: 32x32px o 64x64px

El sistema buscará automáticamente:
- `favicon.png` (prioridad)
- `favicon.ico` (fallback)

Si el archivo no existe, se ocultará el ícono sin causar errores.
